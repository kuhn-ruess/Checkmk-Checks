This check can work standalone or in addition to the integrated Checkmk Azure Monitoring. It provides piggyback data in the same way as the included one, so the hosts will see Addional Services.

- firewall ip config
- firewall policy
- firewall general
- vpn gateway remotepeering
- vpn gateway vpnclient
- vpn gateway ipconfig
- vpn gateway general
- vpn gateway bgp
- connections
- virutalnetworks subnets
- virtualnetworks general
- virutalnetworks peerings
